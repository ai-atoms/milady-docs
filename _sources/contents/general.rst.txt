The present documentation describes the functionalities of the ``MILADY 2.0``
package. The numerical development of ``MILADY`` package started in 2015
at SRMP, CEA Saclay.

The main contributors of this package are in chronological order M.-C. Marinica, W. Unn-Toc
(post doc SRMP, 2015-2016), A. M. Goryaeva (post doc SRMP, 2018 - 2020;
researcher 2021-present), C. Lapointe (PhD SRMP, 2018-2021), J. Baima
(post doc SRMP 2019 - 2021), A. Allera (2022 - present) and software support from Anida Khizar and 
Christian Van Wambeke from LGLS (CEA Saclay).   
The present architecture is designed by MCM and AMG.  The link with ``LAMMPS`` is written by
`Thomas D. Swinburne <https://tomswinburne.github.io/>`_ (CNRS Researcher, CINaM, Marseille)

MiLaDy
------

The ``MILADY`` package corresponds to a traditional machine learning package
with three levels:

- The reduction of the dimensionality and the
  definition of the feature space, using atomic descriptors
- The training strategies of the various machine learning algorithms and
  finally
- The machine learning algorithms.
